# Shopware 6 Default ddev Theme

## Features

Here are the basic features that are implemented in the default theme.

### Storefront 

#### JS-Plugins

- Custom select styling
- Show country state always

#### SCSS

- Functions:
    - Dynamic button styling
    - Basic color scales
    - px to rem function
- Mixins:
    - Layout container
    - Image position
    - Icon selector
    - Primary/secondary card styles
- Varables:
    - Colors
    - Fonts
    - Layouts, spacing, borders, shadows
    - Transition colors
- Basic styling and some resets

#### Twig-Templates
- Basic sticky header

#### Snippets

- Formal and informal german language snippets
